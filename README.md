# DNA Sequence Analyzer

A simple Python tool for analyzing DNA sequences.

## Features

- Calculate DNA sequence length
- Count nucleotide bases (A, T, C, G)
- Calculate GC content
- Generate reverse complement

## Technologies

- Python 3
- Bioinformatics concepts
- Command-line interface (CLI)

## Usage

Run:

```bash
python dna_analyzer.py
```

## Example

Input:

```text
ATGCGCTA
```

Output:

```text
===== DNA Sequence Analyzer =====

Sequence:
ATGCGCTA

Length:
8 bp

Base Counts:

A: 2
T: 2
C: 2
G: 2

GC Content:
50.00%

Reverse Complement:
TAGCGCAT
```
